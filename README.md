# phonegap-example-ember

This is an an example app that runs [ember.js](http://emberjs.com/) on [PhoneGap](http://phonegap.com/).

	git clone https://github.com/dhulihan/phonegap-example-ember
	cd phonegap-example-ember
	phonegap serve
